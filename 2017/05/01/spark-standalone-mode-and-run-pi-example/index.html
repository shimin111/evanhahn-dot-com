准备工作

这里主要配置Spark Standalone，因此hadoop无关紧要。首先将所需要的Spark软件拷贝至各个ubuntu主机上。并且解压，将解压后的文件夹移至/usr/local/目录下(放在任何地方均可，我这里属于个人偏好)，每个主机都需要完成这一步骤。
